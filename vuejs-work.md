# 트렐로 개발로 배우는 Vuejs, Vuex, Vue-Router 프론트엔드 실전 기술

## 개발 순서
### 1. 요구사항 정리(개발 범위 정리)
### 2. 기술조사
### 3. 기능구현
* 인증 : 로그인, 로그아웃
* 보드 작업
    |기능| 생성| 조회| 수정| 이동| 삭제|
    |:---:|:---:|:---:|:---:|:---:|:---:|
    |보드|O|O|타이틀, 색상|X|O|
    |리스트|O|O|타이틀|O|O|
    |카드|O|O|타이틀, 설명|O|O|
        
### 기본플로우
1. 홈페이지 접속
2. 로그인 페이지
3. 홈페이지 리다이렉트
4. 보드 목록 조회
5. 보드 생성
6. 보드 조회 : Todo, Doing, Done
7. 카드 생성, 카드 상세 조회, 카드 수정
8. 카드 이동
9. 카드 삭제
10. 보드 세팅 : 색상
11. 보드 삭제

### 추가 기능
1. 보드 수정 : 색상 변경
2. 보드 수정 : 타이틀 변경
3. 리스트 생성
4. 리스트 수정 : 타이틀 변경
5. 리스트 이동
6. 리스트 삭제

## 라우팅
> 서버 라우팅 : 주소가 변경될 때, 화면이 갱신됨  
> 브라우져 라우팅 : 화면이 갱신되지 않고, 화면에 필요한 데이터만 서버에 따로 요청함
